# Louvain-Community-Detection-Twitch-gamers
Project of Big Data Lecture in TOBB ETÜ 
Osman İlge Ünaldı
Aylin Sena Doğan

We implemented the Louvain Community Detection Algorithm on the Twitch user network, identifying nodes within the same community. This allows us to analyze the network in terms of communities and the attributes associated with these communities.
